# TemperatureConverter

## Description:

The Temperature Converter project is a web-based tool developed using HTML, CSS, and JavaScript, allowing users to convert temperatures between Celsius, Fahrenheit, and Kelvin scales. It offers a simple interface where users can input a temperature value in any of the supported units, and the conversion is instantly calculated and displayed. 

## Technologies Used:

* HTML: Utilized for creating the structure of the web page, including input fields and dropdown menus for selecting temperature units.
* CSS: Employed for styling the user interface, ensuring a visually appealing and intuitive design.
* JavaScript: Implemented for dynamic functionality, including temperature conversion calculations based on user input and displaying the results.

## How to Use:
* Open the web application in a compatible browser: https://fabulousfoxx.github.io/TemperatureConverter/
* Enter the temperature value you want to convert into the provided input field.
* The converted temperature will be automatically calculated and displayed on the screen.
